#TrigonFinal
Все файлы представленные в репозетории являются собственностью команды "Trigon" и представлены для участия в финале конкурса "mpit2021". Использование файлов разрешается только для проверки работы в конкурсе написаном выше.
###
Для тестирования игры перейдите в папку gameBug и откройте файл New Unity Project (1).exe
###
Папки
Assets, Library, Logs, Packages, ProjectSettings, UserSettings - папки самого unity project для работы с проектом.
gameBug - папка с билдом игры, готовый к игре.
###
Управление
w,a,s,d- клавиши вперёд, влево, назад, вправо.
Клавиша X - Взаимодействие (Работает только в лобби (Стартовая локация) ).
Клавиша Z - Пролистывание диалога (Работает только на уровнях обучений).
Клавиша escape - Открытие/закрытие меню.
Управление камеры - мышка/TouchPad
###
Добавленные библиотеки
UnityEngine.UI - библиотека взаимодействия с Canvas
UnityEngine.SceneManagement - библиотека для работы со сценами.
###
Основные фичи и механники проекта и их реализация
!Далее представленно основное описание фич и механник проекта, более подробное описание можно найти в самом коде, в комментариях.!

Открытие/закрытие меню - Скрипт MenuCntrl. При нажатии ecs TimeScale становится равным 0 (время и действия в игре останавливаются). При повторном нажатии esc TimeScale = 1 (Время в игре возобновляется).

Пролистывание диалогового окна/смена заданий - Скрипт TextMas. Весь диалог хранится в определённом порядке в массиве txt. При нажатии кнопки "z" текст сменяется на следующий по порядку в массиве. Текст сменяется до достижения определённого эдемента массива, после которого появляются элементы ввода задания. Текст нельзя сменить, пока задание не будет выполнено. При достижении конца массива игра "блокирует" дальнейшее перелистование текста. При успешном выполнении задания, массив с заданиями "перелистывается" на следующее задание в массиве.

Прыжки по сценам и выход - Скрипт SceneContr. При нажатии на кнопку выхода, вызывается событие EndGame, при его активации приложение закрывается. При нажатии кнопки смены сцены, вызывается событие NextLvl(номер сцены), в событии присутствует метод LoadScene(номер сцены), загружающий сцены с указанным номером.

Передвижение игрока и камеры - Скрипт play. Скрипт считывает нажатие клавиш и если были нажаты определённые клавиши(Смотрите раздел управление), он запускает движение модели или повороты камеры. На уровне обучения Клавиши движения не работают, но работает движение камеры.

Взаимодействие игрока с CheckPoint'ами - Скрипт trying. При попадании модели игрока в определённый checkpoint, срабатывает триггер OnTriggerEnter, который запускает текст в canvas. Далее идёт проверка, если игрок находится на checkpoint'е и нажата определённая кнопка(Смотрите раздел управление), то происходит смена сцены.
